# Brain Health and Chiropractic - Typography

## Primary Font: Playfair Display

**Usage:** Headlines, hero text, section titles, h1-h3

**Characteristics:**
- Elegant serif typeface
- High contrast strokes
- Professional, trustworthy feel

**Weights Used:**
- 400 (Regular) - Subheadings
- 600 (Semi-Bold) - Section titles
- 700 (Bold) - Main headlines

**Google Fonts Link:**
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;600;700&display=swap" rel="stylesheet">
```

**CSS:**
```css
font-family: 'Playfair Display', Georgia, serif;
```

---

## Secondary Font: Open Sans

**Usage:** Body text, navigation, buttons, captions, form fields

**Characteristics:**
- Clean, modern sans-serif
- Highly readable at all sizes
- Friendly and approachable

**Weights Used:**
- 300 (Light) - Large body text, quotes
- 400 (Regular) - Standard body text
- 500 (Medium) - Navigation, labels
- 600 (Semi-Bold) - Buttons, emphasis
- 700 (Bold) - Strong emphasis

**Google Fonts Link:**
```html
<link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

**CSS:**
```css
font-family: 'Open Sans', -apple-system, BlinkMacSystemFont, sans-serif;
```

## Font Pairing Guidelines

| Element            | Font             | Weight | Size (Desktop) |
|--------------------|------------------|--------|----------------|
| Hero Headline      | Playfair Display | 700    | 3.5rem (56px)  |
| Section Title      | Playfair Display | 600    | 2.5rem (40px)  |
| Card Title         | Playfair Display | 600    | 1.5rem (24px)  |
| Navigation         | Open Sans        | 500    | 0.9375rem (15px)|
| Body Text          | Open Sans        | 400    | 1rem (16px)    |
| Button Text        | Open Sans        | 600    | 1rem (16px)    |
| Caption/Meta       | Open Sans        | 400    | 0.875rem (14px)|
| Small Text         | Open Sans        | 400    | 0.75rem (12px) |

## Downloading Fonts (Offline Use)

If you need the actual font files for design software (Figma, Photoshop, etc.):

1. Visit https://fonts.google.com/specimen/Playfair+Display
2. Click "Download family"
3. Install the .ttf files on your system

Repeat for Open Sans:
1. Visit https://fonts.google.com/specimen/Open+Sans
2. Click "Download family"
3. Install the .ttf files on your system
